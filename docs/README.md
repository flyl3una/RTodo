# RTodo 文档目录

本目录包含 RTodo 项目的所有设计文档和任务清单。

## 文档列表

| 文件 | 描述 |
|------|------|
| [design.md](./design.md) | 界面设计方案，包含整体布局、页面设计、组件设计、数据模型和 API 规范 |
| [tasks.md](./tasks.md) | 开发任务清单，按阶段划分的所有开发任务和里程碑 |

## 快速导航

### 设计文档 (design.md)

包含以下内容：
- **整体界面布局**：三栏式布局设计
- **核心页面设计**：侧边栏、任务列表、详情面板、统计页、设置页
- **关键组件设计**：任务卡片、任务组管理、标签选择器、时间选择器、步骤编辑器
- **数据模型**：SQLite 表结构和 TypeScript 类型定义
- **Tauri Command API**：前后端通信接口定义
- **编码规范**：项目开发的编码标准

### 任务清单 (tasks.md)

包含以下内容：
- **阶段 1**：项目初始化（目录结构、依赖安装）
- **阶段 2**：数据库层（表结构、repository）
- **阶段 3**：Rust 数据模型
- **阶段 4**：Tauri Commands
- **阶段 5-6**：前端基础框架和 API 层
- **阶段 7-12**：前端状态管理和组件开发
- **阶段 13-14**：前端页面和 Tauri 配置
- **阶段 15-16**：测试和优化

## 技术栈

- **后端**: Rust + Tauri 2.0 + SQLite
- **前端**: Vue 3 + TypeScript + Vite
- **UI**: Element Plus + TailwindCSS
- **状态管理**: Pinia
- **构建**: pnpm + cargo

## 贡献指南

1. 在开发新功能前，请先阅读 [design.md](./design.md) 了解整体架构
2. 在 [tasks.md](./tasks.md) 中查找要实现的任务
3. 遵循编码规范进行开发
4. 完成后更新任务清单中的进度
